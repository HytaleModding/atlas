# Code of Conduct

Atlas follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html). Be kind, assume good faith, no harassment, no personal attacks.

This applies in the GitHub repo (issues, PRs, discussions) and any HytaleModding community space where Atlas is being discussed.

## Reporting

Direct message the project lead [Vibe Theory](https://github.com/vibetheory) on GitHub or Discord, or report through HytaleModding's mod team. Reports stay private. Outcomes range from a private warning to a permanent ban depending on severity.

The project lead and HytaleModding moderators have final say on enforcement.
